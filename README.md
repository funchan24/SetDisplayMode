## SetDisplayMode

*适用于 联想小新 PRO16 GTX1650。程序每2分钟检查一次电脑是否接通电源、电池余量，如果电脑接通电源，屏幕亮度调为100%，屏幕刷新率调为120；如果电脑使用电池供电，屏幕亮度根据电池余量从80%逐步降至50%，屏幕刷新率调为60。*

### 用法

* 双击 **start.bat** 启动程序
* 程序通过在windows启动文件夹创建快捷方式实现后台自动运行
